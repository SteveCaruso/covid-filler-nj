CoVID Filler

A JavaScript Autofill Injection Bookmarklet For CoVID Vaccine Signups

What is CoVID Filler?

A small bookmarklet that holds a JavaScript "injection" which helps volunteers who are signing folks up for CoVID vaccines enter user information into a number of forms faster than they could do so by hand. It works in conjunction with a spreadsheet signup form.

How Can I Install It?

Click and drag the CoVID Injection link into your bookmark bar. That's it.

When you want to get rid of it, delete the bookmark.

How Can I Use It?

Three easy steps:

  1. Copy the line of the spreadsheet that has the user's information you wish to sign up with into your clipboard.
  2. Go to the vaccine portal.
  3. Click on the bookmarklet you put in your bookmarks bar.

Which Vaccine Portals Are Supported?

Presently only these portals are supported:

  • Rite Aid
  • CVS

Is Any Spreadsheet Supported?

Right now this is being built and maintained for the InterFaith CoVID-19 Vaccine Signup. This spreadsheet assumes that the data in your clipboard is tab-delimited and that the columns come in a specific order.

However, it can be forked on GitHub and adapted, if you have sufficient coding skills.
